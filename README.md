## 💡 Sample Solution Walkthrough

**Problem 180 — Consecutive Numbers** (Medium)

> Find all numbers that appear at least three times consecutively.

```sql
SELECT l1.num AS ConsecutiveNums
FROM logs l1, logs l2, logs l3
WHERE l1.id - l2.id = 1
  AND l2.id - l3.id = 1
  AND l1.num = l2.num
  AND l2.num = l3.num
GROUP BY l1.num;
```

*Approach:* Self-join the `Logs` table three times and match rows where IDs are consecutive and values are identical. The `GROUP BY` removes duplicates.

**Problem 550 — Game Play Analysis IV** (Medium)

> Find the fraction of players who logged in again the day after their first login.

```sql
SELECT ROUND(SUM(player_login) / COUNT(DISTINCT player_id), 2) AS fraction
FROM (
    SELECT player_id,
           DATEDIFF(event_date, MIN(event_date) OVER (PARTITION BY player_id)) = 1 AS player_login
    FROM Activity
) AS t;
```

*Approach:* Uses a window function to compute each player's first login date, then checks if the next day's login exists using `DATEDIFF`. Aggregates the boolean result.
